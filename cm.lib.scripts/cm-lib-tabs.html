CM.Tabs

Analytics

A analytics attribute data-tab-tracker can be placed on the tab where we want tracking

ex: <a href="#tab_0-1" data-tab-tracker='{"DCS.dcsuri":"lorem_tab2","wt.ac":"aze"}'>tab 0-1</a>

note the single quote surrounding to have valid JSON
